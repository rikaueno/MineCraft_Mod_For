🧱 MOD開発（ブロック）

ブロック追加の手順（Minecraft 1.20.1 / Forge）

このページでは、Minecraft 1.20.1 の Forge 環境で、 新しいブロックを追加する手順をまとめています。

① ブロック用の Java ファイルを作成

src/main/java の中に、以下のようなブロッククラスを作成します。


public class MyBlock extends Block {
    public MyBlock() {
        super(Properties.of()
            .strength(3.0f)
            .requiresCorrectToolForDrops());
    }
}

② ブロックを登録する

ModBlocks.java を作成し、DeferredRegister を使って登録します。


public class ModBlocks {
    public static final DeferredRegister BLOCKS =
        DeferredRegister.create(ForgeRegistries.BLOCKS, "modid");

    public static final RegistryObject MY_BLOCK =
        BLOCKS.register("my_block", MyBlock::new);
}

③ ブロックアイテムを登録する

ブロックをインベントリに表示するために、ブロックアイテムも登録します。


public class ModItems {
    public static final DeferredRegister ITEMS =
        DeferredRegister.create(ForgeRegistries.ITEMS, "modid");

    public static final RegistryObject MY_BLOCK_ITEM =
        ITEMS.register("my_block",
            () -> new BlockItem(ModBlocks.MY_BLOCK.get(),
                new Item.Properties()));
}

④ JSON を追加する（モデル・ブロックステート）

src/main/resources に以下の JSON を追加します。

📁 blockstates/my_block.json


{
  "variants": {
    "": { "model": "modid:block/my_block" }
  }
}

📁 models/block/my_block.json


{
  "parent": "block/cube_all",
  "textures": {
    "all": "modid:block/my_block"
  }
}

📁 models/item/my_block.json


{
  "parent": "modid:block/my_block"
}

⑤ テクスチャを追加する

📁 textures/block/my_block.png を追加します。

⑥ 起動して確認する

Gradle → forgegradle runs → runClient を実行し、ブロックが追加されているか確認します。

これでブロック追加は完了です！
