🛠 開発環境構築（Minecraft 1.20.1）

このページでは、Minecraft 1.20.1 用 MOD を開発するための環境構築手順をまとめています。 Java の準備から Forge MDK のセットアップ、IntelliJ でのプロジェクト起動までを順番に説明します。

① Minecraft 1.20.1 用インストーラーのダウンロード

以下のリンクから、1.20.1 用の Installer 一式をダウンロードします。

➡ Minecraft 1.20.1 Installer（Google Drive）

② Java 17 が入っているか確認

コマンドプロンプト（CMD）を開き、以下を入力します。

java -version

Java 17 が表示されない場合は、Eclipse Temurin などから Java 17 をインストールしてください。

③ IDE（統合開発環境）のインストール

MOD開発には IntelliJ IDEA を推奨します。VSCode でも可能ですが、IntelliJ の方が ForgeGradle との相性が良いです。

④ Minecraft Forge MDK のインストール

Forge 公式サイトから、Minecraft 1.20.1 用の MDK（Mod Development Kit） をダウンロードします。

解凍すると、以下のファイルが含まれています：

  • build.gradle
  • gradlew / gradlew.bat
  • src/main/java（ExampleMod）
  • src/main/resources

⑤ IntelliJ でプロジェクトを開く

IntelliJ を起動し、以下の手順で Forge MDK を開きます。

  1. 「Open」から MDK フォルダを選択
  2. 「Trust Project」を押す
  3. JDK を Java 17 に設定
  4. Gradle タブ → genIntellijRuns を実行
  5. runClient を実行して Minecraft が起動するか確認

ここまでできれば、MOD開発の準備は完了です。
