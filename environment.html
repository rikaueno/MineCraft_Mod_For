開発環境構築（Minecraft 1.20.1）

このページでは、Minecraft 1.20.1 用 MOD を開発するための環境構築手順をまとめています。

① Minecraft 1.20.1 用インストーラーのダウンロード

以下のリンクから、1.20.1 用の Installer 一式をダウンロードします。

➡ Minecraft 1.20.1 Installer（Google Drive）

その時に、今回の開発作業で使用するフォルダを作成します。

② Java 17 が入っているか確認

コマンドプロンプトを開き、以下を入力します：

java -version

Java 17 が表示されない場合は、①の中のzulu17.42.19-ca-jdk17.0.7-win_x64.msiからインストールしてください。

③ IDE（統合開発環境）のインストール

IntelliJ IDEA を①の中のidea-2026.1.3.exeからインストールします。

④ Minecraft Forge MDK のインストール

①の中のforge-1.20.1-47.4.13-mdk.zipから解凍します。

⑤ IntelliJ でプロジェクトを開く

  1. IntelliJ → Open → MDKフォルダを選択
  2. Trust Project
  3. JDK を Java17 に設定
  4. Gradle → genIntellijRuns を実行
  5. runClient で起動確認
